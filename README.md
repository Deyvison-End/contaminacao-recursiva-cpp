# Contaminação Recursiva em C++

Projeto desenvolvido em C++ para simular a propagação de contaminação em uma matriz.

## Objetivo

Dada uma matriz contendo:

- T = área contaminada
- A = área saudável

A contaminação se espalha para células adjacentes (cima, baixo, esquerda e direita).

## Conceitos Aplicados

- Recursão
- Matrizes bidimensionais
- Busca em profundidade (DFS)
- Algoritmos iterativos
- Manipulação de vetores (std::vector)

## Implementações

### Solução Recursiva

Utiliza chamadas recursivas para contaminar células adjacentes.

### Solução Iterativa

Utiliza laços de repetição até que nenhuma nova célula seja contaminada.

## Tecnologias

- C++
- STL (Standard Template Library)
